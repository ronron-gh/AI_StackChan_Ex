# Baloo - StackChan CoreS3

An English-first AI companion project for M5Stack CoreS3.

## Current Status

**Phase 0**: English Foundation & Cleanup (in progress)

## Hardware

- M5Stack CoreS3
- Dual servos for movement

## Development Phases

| Phase | Focus                              | Status      |
|-------|------------------------------------|-------------|
| 0     | English Foundation & Cleanup       | In Progress |
| 1     | Real-Time Voice Pipeline           | Planned     |
| 2     | Personality & Teaching Capabilities| Planned     |
| 3     | Animations & Expressiveness        | Planned     |
| 4     | Polish & Additional Features       | Planned     |

## Collaboration

- Work is done iteratively, one task at a time.
- All code changes, prompts, and documentation are provided in English.
- The maintainer handles building, flashing, and physical testing.
